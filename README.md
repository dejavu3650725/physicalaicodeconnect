# 피지컬 AI 코드 커넥트 (Physical AI Code Connect) - 초등 컴퓨팅 사고력(CT) 융합 피지컬 코딩 설계 및 피드백 플랫폼

학생들의 창의적 아이디어(자연어)를 학교 교실에서 가장 널리 쓰이는 '엔트리 햄스터 로봇'의 실제 블록코딩 알고리즘과 실행 가능한 파이썬 코드로 실시간 설계해 주고, AI 교사의 개별화된 피드백을 통해 아이디어를 블록으로 즉각 업데이트해 주는 융합 에듀테크 플랫폼 '피지컬 AI 코드 커넥트'를 제출합니다. 본 서비스는 교사 개발자 해커톤 성장형 연수 이수 과정에서 개발을 시작했고, 이후 검토를 반복하며 고도화했습니다.

## 데모 사이트 안내 (API 키 필요)
본 서비스는 사용자의 안전한 API 관리를 위해 브라우저 로컬 환경의 환경 변수(.env)에 입력된 구글 API(Gemini API) 키를 기반으로 작동합니다. API 연동을 통해 실시간 블록 생성, 파이썬 코드 변환, AI 교사 피드백의 전 과정을 즉시 체험할 수 있습니다.

## 1. 기획 의도

교실 현장에서 피지컬 컴퓨팅 수업을 할 때 교사와 학생들은 큰 장벽에 부딪힙니다. 학교마다 보급된 하드웨어 교구(햄스터, 알버트, 아두이노 등)의 파편화가 심하고, 교구마다 사용되는 핀 번호, 센서 값의 단위, 명령어 블록이 천차만별이기 때문입니다. 

학생들이 "선생님! 저는 쓰레기를 인식해서 삐 소리를 내며 멈추는 분리수거 로봇을 만들고 싶어요!"라고 창의적인 아이디어를 내더라도, 교구가 가진 센서 한계나 블록 명령어의 복잡함 때문에 코딩 구현 단계에서 아이디어가 꺾이기 일쑤였습니다.

서울특별시교육청 교사연구회(AI·피지컬컴퓨팅융합교육연구회) 회장이자 초등학교 5학년을 지도하고 있는 담임 교사로서 저는 수많은 교실 현장의 애로사항을 직접 목격하며 깊이 고민했습니다. "학생들이 만들고 싶어 하는 로봇의 기능(자연어)을 입력하면, 실제 학교에 있는 교구의 1:1 맞춤형 블록 조립도와 텍스트 코드로 자동 설계해 주는 똑똑한 AI 조력자가 있다면 어떨까?"라는 문제의식에서 이 프로젝트가 출발했습니다. 

단순히 결과만 틱 던져주는 것이 아니라, 실제 엔트리 화면과 완벽히 호환되는 블록 비주얼을 시각화하고, 엔트리의 파이썬 텍스트 코드를 복사-붙여넣기 하여 즉시 블록으로 변환시키는 실천적인 연동 흐름을 구축함으로써 교실 속 피지컬 컴퓨팅 수업의 문턱을 획기적으로 낮추고자 하였습니다.

## 2. 핵심 기능

### 1) 실시간 AI 피지컬 알고리즘 설계 (3단계 수준별 학습)
*   학생이 원하는 프로젝트 주제(예: 분리수거 로봇)를 입력하면, 초등 교육과정 수준에 맞춰 체계화된 3단계 코드를 즉시 설계합니다.
    *   기초 (기본 제어): 하드웨어의 모터와 기초 움직임 중심의 제어
    *   기본 (센서와 데이터): 근접 센서, 바닥 센서 등 다양한 입력 데이터값 기반 제어
    *   심화 (인공지능 융합): 엔트리의 AI 모델(비디오 감지, 이미지 인식 등)과 피지컬 로봇의 제어를 융합한 시나리오

### 2) 햄스터 로봇 하드웨어 최적화 (할루시네이션 차단)
*   AI가 가상의 명령어 핀을 지어내지 않고, 실제 햄스터 로봇 교과 자료에 기반한 정식 센서와 동작 명령어만을 사용하도록 설계했습니다.
    *   바닥 센서 값 감지, 근접 센서 값, 손 찾음? 등 정형화된 햄스터봇 제어
    *   삐 소리 버저 연주, 양쪽 바퀴 모터 속도 제어, LED 켜기/끄기 완벽 매핑

### 3) 실제 엔트리와 동일한 '블록 조립도' 시각화
*   엔트리 공식 블록의 오목이-볼록이 연결부(Notch/Tab)와 고유 카테고리 색상(시작: 초록, 흐름: 하늘, 판단: 파랑, 하드웨어: 청록 등)을 CSS로 웹에 완벽하게 재현했습니다.
*   학생들이 레고 조립 설명서를 보듯 직관적으로 인식하고 엔트리 화면에서 바로 똑같이 드래그 앤 드롭으로 조립할 수 있습니다.

### 4) 엔트리 파이썬(Python) 텍스트 코드 변환 및 클립보드 복사
*   각 수준별로 설계된 알고리즘의 엔트리용 파이썬 구문(예: hamster.move_forward_sec(1))을 실시간 생성합니다.
*   [코드 복사] 버튼을 통해 복사한 뒤, 실제 엔트리 프로그램의 [파이썬 모드]에 붙여넣고 [블록 모드]로 전환하면 웹에서 봤던 블록이 엔트리에 즉시 촥촥 자동 조립되는 놀라운 수업 편의성을 제공합니다.

### 5) 대화형 AI 교사 피드백 & 아이디어 블록 자동 업데이트
*   생성된 알고리즘을 바탕으로 학생이 "여기에 장애물을 만나면 삐 소리를 내며 후진하고 싶어요!"라고 추가 아이디어를 입력하면, 친근한 '선생님' 캐릭터가 칭찬과 조언을 아끼지 않는 맞춤형 피드백을 제공합니다.
*   단순 조언을 넘어, [추가된 블록 코딩으로 확인하기] 버튼을 누르면 학생의 추가 아이디어가 실제로 녹아들어 보완된 새로운 엔트리 블록 조립도와 파이썬 코드가 즉시 화면에 업데이트 및 시각화됩니다.

## 3. 강력한 안전 및 프라이버시 보호

*   API 키 보안: 민감한 Google AI Studio API 키를 하드코딩하지 않고 환경 변수(.env)로 완벽 분리하여, 클라이언트 브라우저 노출 및 불필요한 키 유출을 원천 방지하였습니다.
*   초등학생 맞춤형 안전 필터: Gemini API 내부 안전 필터를 적용하여, 부적절하거나 유해한 프로젝트 키워드 유입 시 원천 차단하고 교육적으로 정제된 피지컬 컴퓨팅 범위 내에서만 알고리즘을 설계하도록 유도합니다.

---

## 4. 기술 스택 (Tech Stack)

*   Frontend: React (Vite), Tailwind CSS & Vanilla CSS (엔트리 블록 그래픽 렌더링 최적화), Lucide React (벡터 아이콘)
*   AI Integration: Google Gemini 3.1 Flash API (실시간 융합 알고리즘 및 구조화된 파이썬 코드 설계, 대화형 SEL 피드백 엔진)
*   Version Control & Deployment: Git, GitHub

---

## 5. 향후 계획 및 로드맵 (Roadmap)

*   다양한 하드웨어 교구 지원: 현재 햄스터 로봇에 최적화된 로직을 학교에서 쓰이는 아두이노(Arduino), 알버트(Albert), 비트브릭(Bitbrick), 코드이노(Codeino) 등으로 확장 선택할 수 있는 '교구 선택 필터' 제공 예정.
*   수업 포트폴리오 관리: 생성된 블록과 파이썬 코드를 이미지나 텍스트 파일로 내려받고, 개인별 수업 로그를 저장할 수 있는 데이터베이스 연동 기능 추가 예정.
*   오프라인 가이드라인 보급: 현장 교사들이 이 툴을 이용해 정보 교과 및 실과 피지컬 컴퓨팅 수업에 즉시 투입할 수 있도록 1차시 시범 수업 지도안 및 학습지 템플릿 동시 보급 예정.

## 6. 기대 효과

피지컬 AI 코드 커넥트는 교사의 정보 수업 준비 및 개별 지도 부담을 획기적으로 줄여주는 동시에, 교실 속 코딩 수업의 높은 기술 장벽을 없애줍니다. 학생들이 자연어로 작성한 상상력이 시각적인 엔트리 블록 조립도를 거쳐 파이썬 텍스트 코드로까지 자연스럽게 연결되고, AI 교사의 피드백을 받아 코드가 즉각 업데이트되는 과정을 통해 학생들은 컴퓨팅 사고력(CT)을 흥미롭게 학습할 수 있습니다. '코딩을 두려워하는 학생도 자신의 아이디어를 피지컬 로봇으로 마음껏 표현하고 실행할 수 있는 주도적이고 포용적인 교실'을 만들어가는 것이 이 프로젝트의 최종 목표입니다.
