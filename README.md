# ★ 피지컬 AI 코드 커넥트 (Physical AI Code Connect) - 초등 CT 융합 피지컬 코딩 설계 및 피드백 플랫폼

학생들의 창의적 아이디어(자연어)를 현장에서 가장 널리 쓰이는 '엔트리 햄스터 로봇'의 실제 블록코딩 알고리즘과 실행 가능한 파이썬 코드로 실시간 설계해 주고, AI 교사의 개별화된 피드백을 통해 아이디어를 블록으로 즉각 업데이트해 주는 에듀테크 플랫폼 '피지컬 AI 코드 커넥트'를 제출합니다. 본 서비스는 AI·피지컬컴퓨팅융합수업과 평가를 고민하는 현장 교사의 치열한 고민에서 탄생했습니다.

---

## 1. 기획 의도

교실 현장에서 피지컬 컴퓨팅 수업을 할 때 교사와 학생들은 큰 장벽에 부딪힙니다. 학교마다 보급된 하드웨어 교구의 파편화가 심하고, 핀 번호나 블록 명령어가 천차만별이기 때문입니다. 학생들이 창의적인 아이디어를 내더라도 교구가 가진 센서 한계나 블록의 복잡함 때문에 구현 단계에서 포기하기 일쑤였습니다. 이에 "학생들이 만들고 싶어 하는 로봇의 기능(자연어)을 입력하면, 실제 학교 교구의 1:1 맞춤형 블록 조립도와 텍스트 코드로 자동 설계해 주는 똑똑한 AI 조력자가 있다면 어떨까?"라는 문제의식에서 출발했습니다. 단순히 텍스트 결과만 던져주는 것이 아니라, 엔트리 화면과 완벽히 호환되는 블록 비주얼을 시각화하고 즉시 구동 가능한 코드로 변환시켜 피지컬 컴퓨팅 수업의 문턱을 획기적으로 낮추고자 하였습니다.

---

## 2. 핵심 기능

### 1) 실시간 AI 피지컬 알고리즘 설계 (3단계 수준별 학습)
학생이 원하는 프로젝트 주제를 입력하면 초등 교육과정 수준에 맞춰 기초(움직임)-기본(센서)-심화(AI 융합) 3단계 코드를 즉시 설계합니다.

하드웨어 한계 돌파(Edge Case) 우회 로직: 학생이 하드웨어 제약을 벗어난 요구(예: '하늘을 나는 햄스터봇')를 할 경우 무작정 오류를 내는 대신, "햄스터봇에는 프로펠러가 없지만, 바퀴를 최고 속도로 돌리며 비행기 이륙 소리를 내보는 건 어때?"와 같이 하드웨어 제약 기반 우회 제안 로직을 적용하여 학생의 상상력과 학습 동기를 보호합니다.

### 2) 햄스터 로봇 하드웨어 최적화 (할루시네이션 원천 차단)
AI가 가상의 명령어 핀을 지어내지 않고, 실제 햄스터 로봇 교과 자료에 기반한 정식 센서(바닥 감지, 근접 센서 등)와 동작 명령어만을 사용하도록 시스템 프롬프트를 튜닝했습니다.

### 3) 실제 엔트리와 동일한 '블록 조립도' 시각화
엔트리 공식 블록의 오목이-볼록이 연결부(Notch/Tab)와 고유 카테고리 색상(시작, 흐름, 판단, 하드웨어 등)을 CSS로 웹에 완벽하게 재현했습니다. 레고 조립 설명서를 보듯 직관적으로 인식하고 엔트리 화면에서 똑같이 조립할 수 있습니다.

### 4) 엔트리 파이썬(Python) 텍스트 코드 변환 및 2차 검증
각 수준별로 설계된 알고리즘의 엔트리용 파이썬 구문을 실시간 생성합니다. 특히 생성된 코드가 엔트리 프로그램에서 복사-붙여넣기 시 에러(Syntax Error)를 일으키지 않도록, 프롬프트 체이닝(Prompt Chaining)과 정규식(Regex)을 통한 2차 검증(Validation) 로직을 거쳐 완벽하게 구동되는 코드만을 반환합니다.

### 5) 대화형 AI 교사 피드백 & 아이디어 블록 자동 업데이트
학생이 추가 아이디어를 입력하면 맞춤형 피드백을 제공합니다. 단순 조언을 넘어, [추가된 블록 코딩으로 확인하기] 버튼을 누르면 학생의 추가 아이디어가 실제로 녹아들어 보완된 새로운 엔트리 블록 조립도와 파이썬 코드가 즉시 화면에 렌더링됩니다.

---

## 3. 강력한 안전 및 프라이버시 보호

*   API 키 보안: 민감한 Google API 키를 하드코딩하지 않고 환경 변수(.env)로 완벽 분리하여 클라이언트 브라우저 노출을 방지하였습니다.
*   초등학생 맞춤형 안전 필터: Gemini API 내부 안전 필터를 적용하여 부적절한 키워드 유입을 차단하고 교육적으로 정제된 피지컬 컴퓨팅 범위 내에서만 작동하도록 유도합니다.

---

## 4. 기술 스택 (Tech Stack)

*   Frontend: React (Vite), Tailwind CSS & Vanilla CSS (엔트리 블록 그래픽 렌더링 최적화), Lucide React
*   AI Integration: Google Gemini 3.1 Flash API (실시간 융합 알고리즘 및 텍스트 코드 설계)
*   Version Control & Deployment: Git, GitHub, Vercel

---

## 5. 기대 효과

명령어 블록 탐색 및 하드웨어 연동 오류 수정에 허비되던 시간을 줄여, 학생 1인당 순수 알고리즘 설계(컴퓨팅 사고력) 및 문제 해결 시간을 약 2배 이상 확보합니다. 코딩을 두려워하는 학생도 자신의 아이디어를 로봇으로 마음껏 표현하고 실행할 수 있는 주도적인 교실을 만들어갈 것입니다.
